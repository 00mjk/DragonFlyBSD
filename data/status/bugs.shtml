The project welcomes all bugs reports and tries to make bug reporting as painless an affair as possible by providing several ways to report them.

  • Email: bugs at dragonflybsd.org (mailing list)
  • News: nntp://nntp.dragonflybsd.org/dragonfly.bugs (NNTP)
  • Web: bugs.dragonflybsd.org
  • Non-disclosable security issues: security-officer at dragonflybsd.org

Note: The mailing list, news, and web interface all relate, so for example, an email to the mailing list will be replicated to the newsgroup AND show up in the web bug tracking interface. (The web interface does not post new reports back, however.) We suggest using mail or news, so that issues can be discussed in public.

Bug reports are archived permanently. Please be succinct but detailed when writing your report. The best bug reports are those that include patches, and fit with the goals of the project.
