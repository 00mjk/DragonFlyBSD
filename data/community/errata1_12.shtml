Errata Page for DragonFly 1.12.x

General note on errata: We do not roll a release CD every time we bump the sub-version. The standard upgrade method is to build from sources. The build manual page gives you information on how to upgrade your system from sources. See the download page for information on tracking changes with cvsup.

1.12.1 Errata

System included utilities

    A vendor bug in sendmail that was thought to have been fixed still had issues. A new sub-version had to be rolled with the corrected sendmail.

1.12.0 Errata

CD installation issues

    The CD installer considers the CD itself a hard disk and gives you the option of partitioning it, installing to it, putting bootblocks on it, etc. This obviously won't work and it is a minor annoyance. Simply uncheck the related boxes during the installation process.
