The DragonFly project has several mailing lists for discussion.

To subscribe to a mailing list, send an email to one of the following addresses listed below with the word 'subscribe' in the body. If you need additional help please read bestserv1.html, then send a subscription request to the appropriate list email address as shown below. Please use the test list for experimentation and remember to unsubscribe from it when you are done (unless you want to see other people's test messages!). Messages that include HTML sent to any of the lists will be blocked as spam.

  • Bug Reporting - bugs-request@lists.dragonflybsd.org
  • General/User - users-request@lists.dragonflybsd.org
  • Kernel Related - kernel-request@lists.dragonflybsd.org
  • Patch Submissions - submit-request@lists.dragonflybsd.org
  • Commit Messages - commits-request@lists.dragonflybsd.org
  • Documentation Related - docs-request@lists.dragonflybsd.org
  • HAMMER Filesystem Ports - hammer-request@lists.dragonflybsd.org (NEW LIST)
  • Test Postings - test-request@lists.dragonflybsd.org
These mailing lists may also be accessed through newsgroups or nntp. Posts to newsgroups will be forwarded to the appropriate mailing list and vice versa. When testing newsgroup access, be aware that posts may be delayed up to 5 minutes. Please use the 'test' newsgroup for any test postings / experimentation.

  • Bug Reporting - news://nntp.dragonflybsd.org/dragonfly.bugs
  • Non-kernel General/Users Related - news://nntp.dragonflybsd.org/dragonfly.users
  • Kernel Related - news://nntp.dragonflybsd.org/dragonfly.kernel
  • Patch Submissions - news://nntp.dragonflybsd.org/dragonfly.submit
  • Commit Messages - news://nntp.dragonflybsd.org/dragonfly.commits
  • Documentation Related - news://nntp.dragonflybsd.org/dragonfly.docs
  • HAMMER Filesystem Ports - news://nntp.dragonflybsd.org/dragonfly.hammer
  • Test Postings - news://nntp.dragonflybsd.org/dragonfly.test

There are several different mailing list archives available:

  • leaf.dragonflybsd.org/mailarchive/ (official, all groups)
  • marc.theaimsgroup.com (all groups except docs and test, in BSD section)
  • Gmane.org (kernel only)
